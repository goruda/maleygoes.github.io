5.2 Logarithms and Exponential Models

Why We Care

One of the main applications of logs is solving exponential equations, or problems involving exponential functions. We will see that here!

An example of this would be: We historically have been able to solve the Amount/Population for a given time. Now, we'd be able to find the Time, given an amount or population we're interested in. This is the inverse!

What We'll Learn

  • Solving for the input when its inside the exponent.
  • Converting between \(Q=ab^t\) and \(Q=ae^{kt}\)
  • When we can't solve with logs

What do we need to remember to learn this?

  • You will want to remember the form for an exponential function \( f(t)=a(b)^t =a(1+r)^t \) and try to see why it is equivalent to \( a(e)^{kt} \)
